# Meteor Snippets for Sublime Text

Some Snippets for working with the Meteor[http://meteor.com] Framework. (WIP)

### Codes

__tp__
```
<template name="main">
	...
</template>
```

__mr__
```
Meteor.render(function () {
	...
});
```

__mp__
```
Meteor.publish("name", function () {
	...
});
```

__if__
```
{{#if guyIs "male"}}
	...
{{/if}}
```

__ea__
```
{{#each player}}
	...
{{/each}}
```
